Necesito que me hagas una plantilla de una pagina web
de la tienda de venta de ropa vintage, usando css, html, javascript y php si sea necesario. Diseña la pagina usando color marron, beige, negro,rojo y blanco.
En todas paginas debe aparecer el logo de la tienda arriba a la izuierda, o arriba en el medio en el header.
En la pagina principal debe aperecer logo grande de la tienda que se mueva al bajar la pagina.
La pagina debe cumplir los siguientes requisitos:

- La arquitectura del sitio tiene que ser idéntica en todos los documentos.

- El proyecto tiene que tener en todos sus documentos html, un menú desplegable.

- La página principal, que se denomina index, tiene que tener un slider o carrusel de 
  imágenes (5 imágenes), que realicen algún movimiento o animación. Realiza este slider en la pagina principal, haciendo un slider de nuevo stock.

 - Hay que diseñar la pagina utilizando mediaqueri, atendiendo a dos tamaños de pantalla:
	- Width, máximo 1300 px, disposición para PC
	- Width, máximo 600px, disposición para móvil

 - Tiene que haber un formulario con los siguientes elementos:
Este formulario puedes realizar en la pagina de contacto.

   	- Mínimo dos cajas de texto
	- Lista select desplegable
	- Área de texto
	- Dos botones submit y reset
	- 3 casillas checkbox
	- 3 radiobutton
- Cuando se pulsa el botón de enviar, un archivo .php, procesa los datos y recibe las 
variables.

- Al entrar en la pagina, se debe avisar al usuario con un mensaje en pantalla del uso de las cookies,  de tal forma, que si el usuario pulsa sobre el mensaje, este mensaje desaparece.

- Los elementos de formulario tienen que tener, validación con javascript para 
comprobar
	- No están vacías las cajas
	- No hay caracteres del tipo:$,@,#
	- No comienzan con un número
